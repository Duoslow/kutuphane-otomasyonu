# kutuphane-otomasyonu
Basit php mysqli kütüphane otomasyonu üyelik sistemli
### Çalışan özellikler
-üyelik sistemi  
-admin özel yöneyim sayfası  
-kitap ekleme  
-kitap fotosu ekleme  
-kategori ekleme  
-yazar ekleme  
-yayınevi ekleme  
### Yapılacaklar
-üyelik sistemi geliştirmesi  
-admin paneli iyileştirmesi  
-düzgün query oluşturma  
-spagetti koddan kurtulma
